Ramda 函数库

比较运算

  1. gt : 判断第一个参数是否大于第二个参数; x1 > x2

  2. gte : 判断第一个参数是否大于等于第二个参数; x1 >= x2

  3. gt : 判断第一个参数是否小于第二个参数; x1 < x2

  4. gte : 判断第一个参数是否小于等于第二个参数; x1 <= x2

  5. equals : 比较两个值是否相等（支持对象的比较）

  6. eqBy : 比较两个值传入制定函数的运算结果是否相等

数学运算

  1. add : 返回两个值的和

  2. subtract : 返回第一个参减第二个参的差

  3. multiply : 返回两个参数的积

  4. divide : 返回第一个参数除以第二个参数的商

逻辑运算

  1. either : 接收两个函数作为参数，只要有一个返回 true , 就返回 true ,否则返回 false 。类比于 ||运算

  2. both : 接收两个函数作为参数，只有它们都返回true，才返回true，否则返回false 。类比于 &&运算

  3. allPass : 接收一个函数数组作为参数，只有所有函数都返回 true 时，才返回 true，否则返回false

字符串

  1. split : 按照指定分隔符将字符串拆分为一个数组

  2. test : 判断一个字符串是否匹配给定的正则表达式

  3. match : 返回一个字符串的匹配结果

函数

函数的合成

  1. compose : 将多个函数合并成一个函数，从右到左执行

  2. pipe : 将多个函数合并成一个函数，从左到右执行

  3. converge : 接收两个参数，第一个参数是函数，第二个函数是函数数组。先用函数数组处理传入的值，再将处理结果返回给第一个参数。

柯里化

  1. curry : 将多参数的函数，转换成单参数的形式。
